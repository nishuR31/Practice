Right-Side Sliding Menu with Toggle

Challenge Overview: This challenge is about creating an interactive right-side sliding menu for a website. The main goal is to implement a user-friendly experience that allows users to open and close the menu by clicking a button. Additionally, the menu should automatically close when a user clicks anywhere outside the menu area.
